字符串方法

string
标签 demo 描述
charAt() str.charAt(1); 返回在指定位置的字符。
charCodeAt() str.charCodeAt(1); 返回在指定的位置的字符的 Unicode 编码。
concat() str.concat(str1); 连接字符串。
fromCharCode() str.fromCharCode(65,66,67); //==> ABC 从字符编码创建一个字符串。
indexOf() str.indexOf('a'); 检索字符串。
lastIndex() str.lastIndex('a'); 从后向前搜索字符串。
match() str.match('a'); 找到一个或多个正则表达式的匹配。
replace() str.replace(/w3c/,'w3school'); 替换与正则表达式匹配的子串。
search() str.search('a'); 检索与正则表达式相匹配的值。
slice() str.slice(5);
str.slice(0,5); 		提取字符串的片断，并在新的字符串中返回被提取的部分。
split() str.split('|'); 把字符串分割为字符串数组。
substr(); str.substr(5);
str.substr(3,7); 		从起始索引号提取字符串中指定数目的字符。
substring() str.substring(3);
str.substring(3,7); 		提取字符串中两个指定的索引号之间的字符。
toLocaleLowerCase()
toLowerCase()		 str.toLocaleLowerCase();
str.toLowerCase(); 		把字符串转换为小写。
toLocaleUpperCase()
toUpperCase()		 str.toLocaleUpperCase();
str.toUpperCase(); 		把字符串转换为大写。
- -
anchor() str.anchor('test'); 创建 HTML锚。
big() str.big(); 用大号字体显示字符串。
blink() str.blink(); 显示闪动字符串。
bold() str.bold(); 使用粗体显示字符串。
fixed() str.fixed(); 以打字机文本显示字符串(等宽字体)。
fontcolor() str.fontcolor('red'); 使用指定的颜色来显示字符串。
fontsize() str.fontsize(7); 使用指定的尺寸来显示字符串。
italics() str.italics(); 使用斜体显示字符串。
link() str.link('http://www.baidu.com/'); 将字符串显示为链接。
small() str.small(); 使用小字号来显示字符串。
strike() str.strike(); 使用删除线来显示字符串。
sub() str.sub(); 把字符串显示为下标。
sup() str.sup(); 把字符串显示为上标。
